# Globe — Travel Bot

AI-ассистент турагентства Душанбе 🌍✈️

## Функциональность

- **Каталог туров**: Дубай, Турция, Египет, Мальдивы, Россия с полной информацией о ценах, дате вылета и условиях
- **AI-помощник**: Отвечает на вопросы о городах, странах, достопримечательностях, кухне через LM Studio (Qwen 2.5 Coder)
- **FAQ**: Визы, оплата, отмена туров, условия для детей, страховка, документы
- **История чата**: Сохраняет контекст разговора для более естественного общения
- **Бронирование**: Быстрый переход к менеджеру через Telegram/WhatsApp

## Команды

- `/start` — главное меню
- `/reset` — очистить историю чата

## Архитектура

- **Backend**: python-telegram-bot (асинхронный)
- **LLM**: LM Studio на localhost:1234 (Qwen 2.5 Coder 7B)
- **Storage**: Память разговора в памяти (dict per user ID)
- **Config**: Переменные окружения через python-dotenv

## Данные туров

Каждый тур содержит:
- Название и эмодзи страны
- Цена, длительность, включение в стоимость
- График вылетов
- Фото из Unsplash
- Советы для путешественников
